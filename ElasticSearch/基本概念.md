# 基本概念

索引  ：含有相同属性的文档集合 （相当于mysql中的database）

类型  ： 索引可以定义已给或多个类型，文档必须属于一个类型 (相当于database的table)

文档  ： 文档是可以被索引的基本数据单位 (相当于table中的一条记录)


RESTFul API

##API基本格式：
http://<ip>:<port>/<索引>/<类型>/<文档id>
常用的http动词
GET/PUT/POST/DELETE
